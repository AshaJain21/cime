e3sm_cime_split is a script in CIMEROOT/scripts/Tools.

$ ./e3sm_cime_split --help

A script to split-off E3SM's cime development and make it into a PR for ESMCI

positional arguments:
  repo           Location of repo to use, default is based on current location
                 (default: None)

optional arguments:
  -h, --help     show this help message and exit
  -d, --debug    Print debug information (very verbose) to file
                 /home/aliceb/CESM/cime/scripts/Tools/e3sm_cime_split.log
                 (default: False)
  -v, --verbose  Add additional context (time and file) to log messages
                 (default: False)
  -s, --silent   Print only warnings and error messages (default: False)
  -r, --resume   Resume split after fixing conflicts (default: False)
  -a, --abort    Abort split (default: False)
  --squash       Do the split as squashy as possible (default: False)
